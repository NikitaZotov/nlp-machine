It is an applied service web-program based on OSTIS technology, named as NLP-service, for solving problems of an automatic lexical, lexico-grammatical, syntactic and semantic analysis of natural language texts and their saving.

Used technologies and means in application:

  • Python and C++ languages as main programming languages to develop its application;

  • OSTIS Technology to store and represent natural language texts, and formal description of a linguistics ontology;

  • NLTK resources to analyse lexical, lexico-grammatical, syntactic and semantic compositions of natural language texts;

  • FLASK and Boostrap frameworks to represent information in a RESTful web-interface;

The application currently allows to solve next linguistics tasks:

  1. Input natural language texts and save their on knowledge base. Each text is represented as a set, the translation of which is its text. To solve search tasks, text must have a title.

  2. Analyse lexical and lexico-grammatical compositions of natural language texts that are accompanied by its tokenizing, lemmatizing and tagging. As a result of text analysis, their lexical structures and lexeme graphs are synthesized.

  3. Represent concrete text lexemes and all lexemes, saving in KB, in a form of table. A utility allows to represent and process all lexemes in the table, where columns are their attributes and count forms, but rows are concrete lexemes. The application module provides CRUD operations on lexemes objects: it allows to generate new ones, update and remove old ones.
