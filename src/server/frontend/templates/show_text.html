Text analysis and synthesizing tools:

  • Show text lexemes table

    An utility allows to represent and process concrete text lexemes in the table, where columns are their attributes and count forms, but rows are concrete lexemes. The application module provides CRUD operations on lexemes objects: it allows to generate new ones, update and remove old ones.

  • Show text lexical structure

    A text lexical structure is a structure that describes the text as a decomposition of words and phrases ordered relative to each other in the order, in which they are located in the text, in accordance with which lexemes are put with their morphological paradigms and attributes.

  • Show text syntactic structure

    A text syntactic structure is a structure that describes the text as a component system the internal nodes of which are word combinations, and the leaves are specific words, while each partition of the parent node under a certain syntactic role has a child node, in accordance with which lexemes are set with their paradigms and attributes.

  • Show text semantic structure

    A text semantic structure is a structure that describes the text as a graph dependencies of its lexemes, which are linked with other lexemes with semantic relations.

  • Process new text

    It is a standard input tool, that allows to input new text and process it with other tools.
